# 🌌 AI Persona Vault
> A chill collection of system prompts, personas, and experimental AI behaviors.

Welcome to the vault. This is a community-driven (or just very chill) repository for exploring how AI can be shaped, steered, and occasionally... unconstrained.

## 📂 Structure
- `/personas`: Character profiles and behavioral templates.
- `/system-prompts`: Core instructions for specific tasks or vibes.
- `/jailbreaks`: Historical and experimental bypass/red-teaming prompts (for research).

## 🧘 How to use
Browse the folders, find a prompt you like, and paste it into your favorite LLM. 

## 🤝 Contributions
Keep it chill. Add your favorites, tweak the existing ones, and share the vibes.
