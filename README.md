## 🧘 The Philosophy of the Vault

The **AI Persona Vault** is built on several core pillars:

1.  **Voice Consistency**: A persona should never "break character" unless explicitly asked.
2.  **Structural Integrity**: System prompts should be robust and resist "drift" during long conversations.
3.  **Educational Value**: Every persona should offer a unique perspective that teaches the user something new about the world or the AI itself.
4.  **Ethical Exploration**: While we include jailbreaks for research, we encourage responsible use and a focus on understanding how these systems work.
